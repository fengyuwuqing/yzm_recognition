# 十分钟玩转图像验证识别

```
本项目采用傻瓜式操作：

直接修改项目的config里的配置 


训练直接运行train.py

测试运行predict.py（测试是传入单个文件名或单个文件）

项目缺点:
    1.模型大
    2.测试速度久
    3.不具有通用性（固定位数）
项目优点:
    1.一键操作
    2.网络结构简单，常见的图形验证码绝大多数都可识别（根据难度不同
    样本数量几百到几万不等就可训练出一个不错的模型）
```
